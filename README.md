## Features

- RealTime Chat app
- Live previews
- Fullscreen mode
- Cross platform
- Simple design
- Image Sharing


## Tech Stack

**Client:** React, CSS

**Server:** chatengine-io


## Run Locally

Clone the project

```bash
  git clone https://github.com/Nitin-Sharma-coder/chatcity.git
```

Go to the project directory

```bash
  cd chatcity
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Deployment

To deploy this project run

```bash
  npm run build
```
Then, just provide the build folder to hosting service.


## Installation

Install chativa with npm
